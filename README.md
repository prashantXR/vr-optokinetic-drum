# A Virtual Reality OptoKinetic Drum (SUI 2025)

This project presents a **Virtual Reality implementation of an OptoKinetic Drum (OKD)**, designed to explore **visual motion stimuli and user response in immersive environments**.

📍 Published at **ACM Symposium on Spatial User Interaction (SUI) 2025**  
📍 Authors: Amaan Ranapurwala, Jialin Zhang, Prashant Rawat, Rina Wehbe, Anil Batmaz, Mayra Donaji Barrera Machuca  

---

## 🧠 Motivation

Optokinetic stimulation is widely used in neuroscience and clinical research to study **visual perception, motion response, and vestibular interaction**.

Traditional setups rely on **physical rotating drums**, which are difficult to control, reproduce, and customize.

This project explores how **Virtual Reality can simulate optokinetic stimuli**, enabling controlled, flexible, and immersive experimental environments.

---

## 🔬 Contributions

- Designed a **VR-based OptoKinetic Drum (OKD) system**  
- Simulated **visual motion stimuli** using immersive environments  
- Enabled controlled manipulation of **stimulus parameters (speed, pattern, direction)**  
- Demonstrated the potential of VR for **perception and neuroscience experiments**  
- Provided a reproducible and scalable alternative to physical setups

## 🧪 Method Overview

- Developed an immersive VR environment simulating rotating visual patterns  
- Participants experienced controlled **optokinetic stimuli**  
- System allowed variation in:
  - rotation speed  
  - visual pattern density  
  - motion direction  
- Observed user response and interaction within the VR setup  

---

## 🛠️ System

- Unity3D  
- OpenXR / Oculus SDK  
- Head-Mounted Display (VR headset)  
- Real-time rendering of motion stimuli
